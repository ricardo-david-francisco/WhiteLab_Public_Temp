# Architecture Decision Records

ADRs capture *why* we made a particular structural choice. Each entry
is short (one page), dated, and immutable once accepted — superseded
records are added rather than rewritten.

Format: `NNNN-short-slug.md` with sections **Context**, **Decision**,
**Consequences**, **Status**.
