# Runbooks

Operational procedures for WhiteLab. Every runbook follows the same
shape:

1. **Purpose.** What is this for, when do you reach for it.
2. **Pre-conditions.** What must be true before starting.
3. **Steps.** Numbered, copy-paste-able.
4. **Verification.** How to confirm it worked.
5. **Rollback.** How to undo it.
6. **Audit trail.** Where to record the action.
